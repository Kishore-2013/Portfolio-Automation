# 🚀 Visme AI Portfolio Generator

**Visme AI Portfolio Generator** is a high-performance automation platform that transforms raw resumes into stunning, data-driven portfolios in seconds. Built for developers and professionals who want a premium digital presence with zero manual effort.

## ✨ Key Features

### 🧠 Intelligent Resume Parsing
Upload any PDF resume and watch our AI (OpenAI/Ollama) extract your skills, experiences, and achievements into structured data automatically.

### 📊 Dynamic Hero Stats
Your portfolio's Hero section metrics (e.g., "5+ Projects", "3+ Years Experience") are dynamically calculated from your resume data, ensuring your digital presence is always up-to-date.

### 🎨 AI Project Illustrations
Every project in your portfolio gets a unique, high-quality 3D isometric illustration generated by **DALL-E 3** based on your specific project description.

### 📥 Real Resume Downloads
Visitors can download your actual original PDF resume directly from your portfolio. The system automatically syncs your latest upload to your live site.

### ⚡ One-Click Deployment
Deploy your generated portfolio to GitHub or Vercel with a single click directly from the dashboard.

## 🏗️ Technical Architecture

The platform is built as a monorepo for maximum scalability and performance:

- **`web/`**: The Creator Dashboard. A Next.js application where users upload resumes and customize their portfolios.
- **`server1/`**: The Automation Engine. Handles AI parsing, image generation, asset orchestration, and portfolio instance management.
- **`server1/templates/`**: Premium React/Vite templates used as the foundation for your generated portfolios.

### Tech Stack
- **Dashboard**: React, Next.js, Framer Motion, Lucide Icons.
- **Backend**: Node.js, Express, Supabase (PostgreSQL).
- **AI/ML**: OpenAI GPT-4o, DALL-E 3, Ollama (Local LLM Support).
- **Deployment**: GitHub API, Vercel API.

## 🛠️ Getting Started

### 1. Prerequisites
- Node.js (v18+)
- Supabase Account
- OpenAI API Key

### 2. Installation
```bash
git clone https://github.com/Kishore-2013/Portfolio-Automation.git
cd Portfolio-Automation
npm install
```

### 3. Environment Setup
Configure your `.env` files in both `/web` and `/server1` based on the provided `.env.example` files. (Refer to the `SETUP_GUIDE.md` for detailed variable mappings).

### 4. Development
Launch both the dashboard and the backend engine:
```bash
npm run dev
```
